É um site que se movimenta utilizando recursos de javascript, imagens recortadas específicas e css (utilizo principalmnte z-index). Fiz esse projeto como um exemplo do que consigo fazer em questão de design e front-end com recursos simples e criatividade.

Meu trabalho

Trabalhei bastante com desenvolvimento de telas funcionais para um sistema de pós-vendas utilizado pelos clientes da empresa para atendimento pós-venda de seus produtos e acionamento de garantia.
Também criei telas para o novo app que será utilizado pelos técnicos dos postos autorizados dos clientes para atender e realizar conserto de produtos, como telas de inventário e pedido de peças, orçamento, login do técnico, rotinas que geram documentos excel com os produtos, peças ou guias de conserto tela de ativação e desativação de postos avançados e funcionários (técnicos dos postos), além de muitos ajustes no front em geral.
Utilizei principalmente PHP, HTML, Javascript e o banco de dados (Postgresql).
